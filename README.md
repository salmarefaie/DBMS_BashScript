# Database_shellScript
## Description
> It is a simple database engine project to make simple operations with databases and tables. 

<a name="desc"></a>
## DataBase Menu
> - Create DataBase
> - List Databases
> - Drop Database
> - Connect to Database
> - Exit 

## Table Menu
> - Create Table
> - List Tables
> - Drop Table
> - Insert Data Into Table
> - Select Data From Table
> - Delete Data From Table
> - Update Data in Table
> - Return To Main Menu
